# DiceRoom

## 概要

システムで指定したダイスを振って記録していくアプリケーションです

## サーバの立て方

誰でも簡単に構築する事ができるような技術を選択して開発されています
以下の手順でデプロイしてください

### Windows

* nodejsをインストールします
* mongodbをインストールします
* gitをインストールします
* 任意のディレクトリ上でgit clone <url>を実行しgithubからプロジェクトを取得します
* mongodbを起動します
* node app app.jsを実行します
* http://localhost:3000/ にアクセスして動作を確認してください
